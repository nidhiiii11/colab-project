# Titanic Dataset - Exploratory Data Analysis (EDA)

This project performs a comprehensive Exploratory Data Analysis on the classic Titanic dataset using Python.

## Key Insights
* **Survival by Class:** Passengers in 1st class had a significantly higher survival rate (62.9%) compared to 2nd class (47.2%) and 3rd class (24.2%).
* **Gender Analysis:** There was a clear disparity in survival, with females having a much higher survival count across all classes.
* **Data Quality:** Used Seaborn heatmaps to identify missing values in columns like `age` and `deck`.

## Libraries Used
* Pandas for data manipulation.
* Seaborn and Matplotlib for data visualization.
* Numpy for statistical analysis.
